## Working principles

- Lead with **trust and clarity**, not hype
- Show the product quickly with the built-in **demo report**
- Be explicit about **privacy boundaries** and **LLM limitations**
- Optimize for **stars, quality feedback, contributor interest, and repeat usage signals**
- Reuse one core launch story across channels, then adapt tone per audience
